# FullStack&Python AI chatbot_Team_Project_MoneyBricks
SpringBoot, React, Python을 이용하고 AWS를 이용해 배포를 한 금융교육사이트


## 프로젝트 소개
사회초년생을 위한 예/적금, 모의주식, 퀴즈를 통한 금융지식 습득을 위주로 포인트를 통한 성취를 습득 할 수 있는 사이트를 구현한 프로젝트입니다.
